Add a new section to `createWeeklySnapshots()` after the existing Frist snapshot block. This creates a kanzlei-wide (non-per-user) snapshot of open Wiedervorlagen for backlog delta history:

```typescript
// Snapshot open Wiedervorlagen per kanzlei (for team dashboard backlog delta)
const kanzleien = await prisma.kanzlei.findMany({ select: { id: true } });

for (const k of kanzleien) {
  const wvCount = await prisma.kalenderEintrag.count({
    where: {
      typ: "WIEDERVORLAGE",
      erledigt: false,
      akte: { kanzleiId: k.id },
    },
  });

  // Use findFirst + create/update pattern because Prisma compound unique
  // with nullable userId has edge cases (PostgreSQL NULL != NULL).
  const existing = await prisma.weeklySnapshot.findFirst({
    where: { model: "Wiedervorlage", weekStart, userId: null },
  });

  if (existing) {
    await prisma.weeklySnapshot.update({
      where: { id: existing.id },
      data: { count: wvCount },
    });
  } else {
    await prisma.weeklySnapshot.create({
      data: { model: "Wiedervorlage", weekStart, userId: null, count: wvCount },
    });
  }
}
```

**2. Create `src/app/api/admin/team-dashboard/route.ts`:**

GET endpoint returning aggregated team metrics JSON. ADMIN-only access.

```typescript
import { NextResponse } from "next/server";
import { auth } from "@/lib/auth";
import { prisma } from "@/lib/db";
import { startOfWeek, subWeeks, format, startOfDay } from "date-fns";
import { de } from "date-fns/locale";

export const dynamic = "force-dynamic";

export async function GET() {
  const session = await auth();
  if (!session?.user || (session.user as any).role !== "ADMIN") {
    return NextResponse.json({ error: "Nicht autorisiert" }, { status: 403 });
  }
  const kanzleiId = (session.user as any).kanzleiId;

  const [questRate, backlogData, bossfightData] = await Promise.all([
    getTeamQuestFulfillmentRate(kanzleiId),
    getBacklogDeltaHistory(kanzleiId, 8),
    getBossfightHistory(kanzleiId),
  ]);

  return NextResponse.json({ questRate, backlog: backlogData, bossfight: bossfightData });
}
```

Implement three helper functions inside the route file (server-only, never exposed to client):

**`getTeamQuestFulfillmentRate(kanzleiId: string): Promise<number>`**
- Get opted-in users: `prisma.user.findMany({ where: { kanzleiId, gamificationOptIn: true, aktiv: true }, select: { id: true, gameProfile: { select: { klasse: true } } } })`
- Return 0 if no opted-in users
- Get active daily quests: `prisma.quest.findMany({ where: { typ: "DAILY", aktiv: true }, select: { id: true, klasse: true } })`
- Get today's completions: `prisma.questCompletion.findMany({ where: { completedDate: startOfDay(new Date()), quest: { typ: "DAILY" }, userId: { in: userIds } }, select: { userId: true, questId: true } })`
- For each user: available = quests where `klasse === null || klasse === user.gameProfile?.klasse`; completed = count matching completions
- Average ratios across users, return `Math.round((totalRate / users.length) * 100)`

**`getBacklogDeltaHistory(kanzleiId: string, weeks: number): Promise<{ dataPoints: Array<{week: string, count: number}>, currentCount: number, trend: "steigend"|"fallend"|"stabil" }>`**
- Get current live WV count: `prisma.kalenderEintrag.count({ where: { typ: "WIEDERVORLAGE", erledigt: false, akte: { kanzleiId } } })`
- Get historical snapshots: `prisma.weeklySnapshot.findMany({ where: { model: "Wiedervorlage", weekStart: { gte: subWeeks(startOfWeek(new Date(), {weekStartsOn:1}), weeks-1) }, userId: null }, orderBy: { weekStart: "asc" } })`
- Build data points with `format(s.weekStart, "dd.MM.", { locale: de })` labels
- Append current week as live data point
- Compute trend from last 2 points: diff > 0 = steigend, diff < 0 = fallend, else stabil

**`getBossfightHistory(kanzleiId: string): Promise<{ activeDamage: number, history: Array<{id, name, spawnHp, currentHp, status, spawnedAt, defeatedAt, totalDamage}> }>`**
- Get bossfights: `prisma.bossfight.findMany({ where: { kanzleiId }, orderBy: { spawnedAt: "desc" }, take: 10 })`
- For each, aggregate damage: `prisma.bossfightDamage.aggregate({ where: { bossfightId }, _sum: { amount: true } })`
- Sum all totalDamage for the overall team metric
- Serialize dates to ISO strings for JSON transport

CRITICAL: No per-user data leaves these functions. Only aggregate numbers.
