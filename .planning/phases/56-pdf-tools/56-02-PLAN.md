**pdf-page-thumbnails.tsx** -- Drag-and-drop page reorder component:
- Props: `{ pageCount: number, onReorder: (order: number[]) => void, selectedPages?: Set<number>, onSelectPage?: (page: number) => void }`
- Renders a grid of page number cards (4 columns, responsive) -- NO actual PDF rendering (too heavy), just numbered cards with page indicators (e.g., "S. 1", "S. 2")
- Each card: 80x100px, border, page number centered, drag handle icon
- Use native HTML drag-and-drop (dragStart, dragOver, drop) -- do NOT add dnd-kit or react-beautiful-dnd (would add a new dependency). Use simple state-based reorder with `onDragStart` capturing the source index, `onDragOver` with `e.preventDefault()`, and `onDrop` splicing the order array.
- Optional selection mode: clicking a page toggles it in `selectedPages` (blue highlight border)
- Export `PdfPageThumbnails` component

**pdf-tools-dialog.tsx** -- Main PDF tools dialog:
- Props: `{ open: boolean, onOpenChange: (open: boolean) => void, dokument: { id: string, name: string, akteId: string }, onComplete: () => void }`
- Uses shadcn Dialog + Tabs component (import from `@/components/ui/dialog` and `@/components/ui/tabs`)
- 5 tabs: Aufteilen (Split), Drehen (Rotate), Komprimieren (Compress), Wasserzeichen (Watermark), Schwaerzen (Redact)
- Tab content for each operation:

  **Aufteilen (Split):**
  - First fetches page count from Stirling-PDF info endpoint (or accepts manual input as fallback)
  - Shows `PdfPageThumbnails` with selection mode for choosing pages
  - Text input for page range (e.g., "1-3,5,7-9") with helper text
  - "Aufteilen" button

  **Drehen (Rotate):**
  - Radio group or segmented buttons: 90, 180, 270 Grad
  - Visual indicator showing rotation direction
  - "Drehen" button

  **Komprimieren (Compress):**
  - Slider or select with 5 levels: Niedrig (1), Mittel (2), Standard (3), Hoch (4), Maximal (5)
  - Helper text: "Hoehere Komprimierung = kleinere Datei, aber moeglicher Qualitaetsverlust"
  - "Komprimieren" button

  **Wasserzeichen (Watermark):**
  - Quick preset buttons: "ENTWURF", "VERTRAULICH", "KOPIE"
  - Custom text input for custom watermark
  - Optional: font size slider (20-60, default 30), rotation slider (0-90, default 45), opacity slider (0.1-1.0, default 0.5)
  - "Wasserzeichen anwenden" button

  **Schwaerzen (Redact):**
  - Checkbox: "DSGVO-Muster anwenden" (pre-checked) -- auto-selects IBAN, Telefon, E-Mail, Steuernummer patterns
  - Individual checkboxes for each DSGVO pattern type
  - Custom text input for additional search terms
  - Warning banner: "Bitte pruefen Sie das Ergebnis vor dem Speichern"
  - "Schwaerzen" button

- Each operation tab has a shared footer:
  - Toggle: "Als neue Datei speichern" (default: checked)
  - Action button (calls `POST /api/dokumente/{id}/pdf-tools` with correct params)
  - Loading state with Loader2 spinner
  - On success: toast.success with file name, call `onComplete()`, close dialog
  - On error: toast.error with German message

- All UI strings in German. Use existing shadcn components (Dialog, Tabs, TabsList, TabsTrigger, TabsContent, Button, Input, Slider, Label, Checkbox, RadioGroup).
- Import icons from lucide-react: Scissors (split), RotateCw (rotate), Minimize2 (compress), Stamp (watermark), ShieldAlert (redact), Loader2.
