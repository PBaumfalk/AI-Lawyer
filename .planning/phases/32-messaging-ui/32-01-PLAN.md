API response shapes:
```typescript
// GET /api/channels -> { channels: ChannelListItem[] }
// GET /api/channels/[id]/messages?cursor=X&limit=50 -> { messages: MessageListItem[], nextCursor: string | null }
// POST /api/channels -> { channel: { id, name, slug, ... } } (status 201)
// POST /api/channels/[id]/messages -> { message: { ... } } (status 201)
// PATCH /api/channels/[id]/read -> { success: true }
// GET /api/channels/[id]/members -> { members: { userId, userName, userAvatarUrl }[] }
// POST /api/channels/[id]/reactions -> { reaction: { ... } } (status 201)
// DELETE /api/channels/[id]/reactions (body: { messageId, emoji }) -> { success: true }
```
