**GET handler:** List alerts with filters and pagination.

- Auth: `const result = await requireAuth(); if (result.error) return result.error;`
- Parse query params:
  - `typ` (optional): filter by HelenaAlertTyp. Validate against valid set: `FRIST_KRITISCH, AKTE_INAKTIV, BETEILIGTE_FEHLEN, DOKUMENT_FEHLT, WIDERSPRUCH, NEUES_URTEIL`
  - `akteId` (optional): filter by specific Akte
  - `prioritaet` (optional): minimum priority filter. Parse as integer (1-10). When provided, filter alerts with `prioritaet >= parsedValue`. E.g., `?prioritaet=7` returns only high-priority alerts.
  - `gelesen` (optional): `"true"` or `"false"` for read/unread filter
  - `page` (default 1, min 1)
  - `limit` (default 20, min 1, max 100)
- Build WHERE clause:
  ```typescript
  const akteFilter = buildAkteAccessFilter(session.user.id, session.user.role);
  const where: Record<string, unknown> = { akte: akteFilter };
  if (typ && VALID_TYPES.has(typ)) where.typ = typ as HelenaAlertTyp;
  if (akteId) where.akteId = akteId;
  const prioritaetParam = searchParams.get("prioritaet");
  if (prioritaetParam) {
    const minPrio = Math.max(1, Math.min(10, parseInt(prioritaetParam, 10)));
    if (!isNaN(minPrio)) where.prioritaet = { gte: minPrio };
  }
  if (gelesen === "true") where.gelesen = true;
  if (gelesen === "false") where.gelesen = false;
  ```
- Query with pagination:
  ```typescript
  const skip = (page - 1) * limit;
  const [alerts, total] = await Promise.all([
    prisma.helenaAlert.findMany({
      where,
      orderBy: { createdAt: "desc" },
      skip,
      take: limit,
      include: {
        akte: { select: { id: true, aktenzeichen: true, kurzrubrum: true } },
      },
    }),
    prisma.helenaAlert.count({ where }),
  ]);
  ```
- Return: `NextResponse.json({ alerts, total, page, pages: Math.ceil(total / limit) })`

**PATCH handler:** Bulk mark all alerts as read ("Alle gelesen" button).

- Auth check
- Update all unread alerts accessible to the user:
  ```typescript
  const akteFilter = buildAkteAccessFilter(session.user.id, session.user.role);
  const updated = await prisma.helenaAlert.updateMany({
    where: { akte: akteFilter, gelesen: false },
    data: { gelesen: true, gelesenAt: new Date() },
  ```

- Emit badge count update: `getSocketEmitter().to("user:" + session.user.id).emit("helena:alert-badge", { count: 0 })`
- Return: `NextResponse.json({ updated: updated.count })`

```typescript
const fetchAlerts = useCallback(async () => {
  setLoading(true);
  const params = new URLSearchParams();
  if (filters.typ) params.set("typ", filters.typ);
  if (filters.akteId) params.set("akteId", filters.akteId);
  if (filters.prioritaet) params.set("prioritaet", filters.prioritaet);
  if (filters.gelesen !== null) params.set("gelesen", filters.gelesen);
  params.set("page", String(page));
  params.set("limit", "20");

  try {
    const res = await fetch(`/api/helena/alerts?${params.toString()}`);
    if (res.ok) {
      const data = await res.json();
      setAlerts(data.alerts);
      setTotal(data.total);
      setPages(data.pages);
    }
  } catch { /* Non-critical */ }
  finally { setLoading(false); }
}, [filters, page]);
```
