```typescript
// Client emits:
socket.emit("join:channel", channelId)   // Join channel room
socket.emit("leave:channel", channelId)  // Leave channel room
socket.emit("typing:start", channelId)   // Start typing indicator
socket.emit("typing:stop", channelId)    // Stop typing indicator

// Server emits to channel room:
// "message:new" -> MessageNewPayload (from message-service.ts sendMessage)
// "message:edited" -> MessageEditedPayload
// "message:deleted" -> MessageDeletedPayload
// "typing:start" -> TypingPayload (relayed to other members)
// "typing:stop" -> { channelId, userId } (relayed to other members)
// "reaction:added" -> { messageId, emoji, userId, channelId }
// "reaction:removed" -> { messageId, emoji, userId, channelId }
```
