2. **Create PATCH /api/akten/[id]/dokumente/[dId]/mandant-sichtbar** endpoint:
   - Accept `{ mandantSichtbar: boolean }` body
   - Use `requireAkteAccess(akteId, { requireEdit: true })` for RBAC (only ADMIN/ANWALT/SACHBEARBEITER can toggle)
   - Validate the Dokument belongs to the given Akte
   - Update `prisma.dokument.update({ where: { id: dId }, data: { mandantSichtbar } })`
   - When toggling TO `true`, create an AktenActivity:
     ```
     prisma.aktenActivity.create({
       data: {
         akteId,
         userId: session.user.id,
         typ: "DOKUMENT",
         titel: `Dokument "${dokument.name}" fuer Mandant freigegeben`,
         meta: { dokumentId: dId, mandantSichtbar: true },
       }
     })
     ```
     Note: Phase 45 adds `mandantSichtbar` to AktenActivity model. Set it to `true` on this activity so Mandant sees "Neues Dokument freigegeben" in timeline.
   - When toggling TO `false`, do NOT create an activity (Mandant should not see revocations)
   - Return the updated document
   - Use `logAuditEvent()` for both toggle directions
  </action>
